001 : FlexToolsApp

FlexToolsApp is my Bachelor's Degree App. I used RoboFriends as a start but it's an improvement containing a lot more components and functionalities making a perfect app for a big company's tool management called Flextronics Timisoara.

Once you have chosen one of the cards, a new page will display details about it: Name, Category, Rack (Rack number), Date (date the item was added / checked in the Maintenance department) and Status (Available, Unavailable, Maintenance ). Each tool, after 6 months from its addition or from the last maintenance, will blur and will color its edge in yellow, which will determine the staff to send the tool to the Maintenance department.

Even though the database is easy to access, we have supplemented the application with a tool adding function directly from the main page of the application to simplify and improve the Flex Tools App.

By accessing the "back" button we can return to the home page, which is obtained by linking the function of adding a tool with the rest.

This is a card description which you can see pressing any of the cards in the main page.

The "Strapi" database used in this project is a software novelty being a service easy to use and understood by anyone who would need to modify the tools and their description.
